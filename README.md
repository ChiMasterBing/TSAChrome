# Colorveal
Colorveal is a chrome extension that helps the colorblind in browsing the internet.

This chrome extension utilizes filters to manipulate the colors and increase the saturation of color-impared vision. Acting as "color-blindness glasses" on your computer.

There are many different types of color-blindness but they can be seperated into two general types - complete blindness to a color, or imparity to a color. When complete blindness to a color occurs, there is not much that can be done to differentiate that color than adapt them to a different color. However, when color imparity occurs, the image can be manipulated and be seen clearer to the color-blind person.
